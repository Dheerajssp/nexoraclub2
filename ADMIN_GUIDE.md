# 📚 Nexora Club - Admin Guide

## 🎯 Kaise Use Karein (Step-by-Step Guide)

### 1️⃣ Admin Panel Access Kaise Karein

**Step 1: Register/Login**
1. Website par jao: `/join`
2. Apna registration form bharo
3. Password banao (minimum 6 characters)
4. "Submit Application" dabao
5. Automatically login ho jayega

**Step 2: Admin Access**
1. Browser mein jao: `/admin`
2. Agar login hai to admin panel khul jayega
3. Agar login nahi hai to "Join Us" page par redirect hoga

---

### 2️⃣ Apna Event Kaise Add Karein

**Admin Dashboard Par:**

1. **"Add Event" Button** dabao (top right corner)
2. **Form Fill Karo:**
   - **Event Title**: Event ka naam (e.g., "Tech Fest 2024")
   - **Description**: Event ke baare mein detail
   - **Date**: Event ki date select karo
   - **Category**: Select karo (Hackathon, Workshop, Bootcamp, etc.)
   - **Image URL**: Event ki photo ka link paste karo

3. **Image Kahan Se Laye?**
   - **Unsplash**: https://unsplash.com (free professional photos)
   - **Pexels**: https://www.pexels.com (free stock photos)
   - Koi bhi image right-click → "Copy Image Address" → Paste karo

4. **"Create Event"** dabao

✅ **Event Create Ho Gaya!** Ab ye Events page par "Nexora Events" tab mein dikhega.

---

### 3️⃣ Event Edit/Delete Kaise Karein

**Edit Event:**
1. Admin Dashboard par jao
2. "Your Nexora Events" table mein apni event dhundo
3. **Edit icon (pencil)** dabao
4. Changes karo aur save karo

**Delete Event:**
1. Event ke samne **Delete icon (trash)** dabao
2. Confirm karo
3. Event delete ho jayega

---

### 4️⃣ External Events Sync Kaise Karein

1. Admin Dashboard par jao
2. **"Sync External"** button dabao (top right)
3. Wait karo 5-10 seconds
4. External events (Unstop, HackerEarth, Devfolio) automatically add ho jayengi

✅ Ye events **"Events from Top Platforms"** tab mein dikhengi.

---

## 🎨 Event Images Kahan Se Laye?

### Recommended Free Image Platforms:

1. **Unsplash** (https://unsplash.com)
   - Search karo: "hackathon", "technology", "coding"
   - Image par right-click → "Copy image address"
   - URL paste karo form mein

2. **Pexels** (https://www.pexels.com)
   - Same process as Unsplash
   - High-quality free images

3. **Google Images**
   - Search karo
   - "Tools" → "Usage Rights" → "Creative Commons"
   - Right-click → "Copy image address"

### Image URL Example:
```
https://images.unsplash.com/photo-1504384308090-c894fdcc538d?w=800&q=80
```
