## 🎯 **How to Add Your Own Events**

### **Method 1: Through Admin Panel** (Recommended)

1. Go to: `https://your-website.com/admin`
2. Login with your credentials
3. Click **"Add Event"** button
4. Fill the form:
   - **Title**: Your event name
   - **Description**: What's the event about
   - **Date**: When is it happening
   - **Category**: Hackathon/Workshop/Bootcamp/Seminar/Competition
   - **Image URL**: Get from Unsplash.com
5. Click **"Create Event"**

✅ **Done!** Your event will show in "Nexora Events" tab

### **Method 2: External Events Auto-Sync**

1. Go to Admin Panel (`/admin`)
2. Click **"Sync External"** button
3. Wait 5-10 seconds
4. External events from Unstop, HackerEarth, Hack2Skill will be added automatically

## 📸 **Where to Get Event Images?**

### **Free Image Websites:**

1. **Unsplash** (https://unsplash.com)
   - Search: "hackathon", "technology", "coding", "workshop"
   - Right-click on image → "Copy image address"
   - Paste in "Image URL" field

2. **Pexels** (https://www.pexels.com)
   - Same process as Unsplash
   - High-quality free stock photos

## 🛠️ **Editing/Deleting Events**

### **Edit Event:**
1. Admin Dashboard → "Your Nexora Events" table
2. Find your event
3. Click **Edit icon** (pencil)
4. Make changes
5. Save

### **Delete Event:**
1. Find event in admin table
2. Click **Delete icon** (trash)
3. Confirm deletion
